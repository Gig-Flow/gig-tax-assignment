# Tax Computation Assignment

## Inputs
- gross_income = 48000
- expenses = 3200
- miles = 9500
- mileage_rate = 0.70
- se_rate = 0.153
- income_rate = 0.12

## What to Do
Write a function `compute_taxes()` / `computeTaxes()` that:
- Validates inputs (no negatives)
- Calculates mileage deduction, total deductions, net income, SE tax, income tax, total tax, and effective tax rate
- Returns these values or prints them neatly.
