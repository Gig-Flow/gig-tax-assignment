# Tax Computation for U.S. Gig Workers — Software Engineering Assignment

Implement a small **tax computation tool** in **Python or TypeScript**.

## Scenario (2025)
- Gross Income: $48,000
- Expenses: $3,200
- Business Miles: 9,500
- Mileage Rate: $0.70/mile
- Self-Employment Tax: 15.3%
- Income Tax: 12%

## Tasks
1. Mileage deduction  
2. Total deductions  
3. Net income  
4. Self-employment tax  
5. Income tax  
6. Total tax  
7. Effective tax rate

## Deliverables
- Code with input validation
- CLI output showing all computed values
- Short README (5–10 lines about approach)

## How to Submit
1. Fork this repo  
2. Create a branch `feature/solution-<yourname>`  
3. Implement code in **either** `python/` **or** `typescript/`  
4. Run tests and ensure they pass  
5. Open a PR to `main`  
6. CI will automatically test your work
